
### Installing

For install need Python 3.6.8

```
pip3 install > requirements.txt
```

##Environments

set variables:

```
VIRTUAL_ENV = <prodaction>|<development>|<test> 
HOST = <your host>
PORT = <your port>
MONGODB_URI = <URI MongoDB>
```

## Running the tests

```
nose2 -v
```

### Running production


```
python3 app.py runserver
```
